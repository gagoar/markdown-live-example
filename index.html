Markdown

DNS server pipe-backend for PowerDns with particular behavior. You can serve different A records of any domain depending on the query request client. Usefull in Wan and Lan environments.

The Admin Website

This system is provided with a simple and usefull User Interface that allows you to quickly and easily manage it.

Stylesheet: the css structure has three sections, the first one reset the default styles, the second one has tipography and colors definitions. You can modify them without apply any layout changes because you have that propertie's definitions in a separate and last section.
